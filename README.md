# Connectify (Frontend with REACTJS)
This is a social media web application that allows users to create accounts, make friends, add posts, react to others posts, basically, almost every feature a simple social media application can have.

### To run it:
- Make sure you have Nodejs (npm) installed
- In the same directory, type the command in cmd: npm install
- To run, type: npm run dev
- The frontend will be hosted locally on port 5173
